== Description ==

<h3>WPMK FAQ</h3>
This <strong>WPMK FAQ</strong> will add faq page on your website with the help of shortcode, we made very easy shortcode for you will diffrent skins. This plugin is also responsive which means it would work perfectly on desktop, tablet and mobile screen.

= Shortcode with skin and icon =
<pre><code>[wpmk-faq skin="red" icon="fa-pie-chart"]</code></pre> 
<pre><code>[wpmk-faq skin="blue" icon="fa-bell"]</code></pre>
<pre><code>[wpmk-faq skin="green" icon="fa-battery-half"]</code></pre>
<pre><code>[wpmk-faq skin="yellow" icon="fa-skype"]</code></pre>
<pre><code>[wpmk-faq skin="black" icon="fa-whatsapp"]</code></pre>
<pre><code>[wpmk-faq skin="sky" icon="fa-wordpress"]</code></pre>


= Plugin Features =

1. We have set 6 skins and more skin will be add soon.
2. You can use any <strong>font Awesome</strong> icon.
3. Work as part of your theme.
4. Custom Post Type.
5. Sort FAQ Order.
6. Light waight
7. SEO Friendly
8. Responsive

== Installation ==

1. Upload `WPMK FAQ` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Simply download and upload / Installed and active the plugin.
